# 🩺 Glucose Analyzer

> Aplicação em Python para análise de glicose e recomendação de consumo de alimentos com base no teor de açúcar.

---

## 🚀 Sobre o projeto

O **Glucose Analyzer** é um projeto desenvolvido com foco em lógica de programação e boas práticas em Python.
Ele simula uma análise simples de níveis de glicose e sugere quais alimentos são recomendados ou não, com base no percentual de açúcar.

---

## 🎯 Objetivos

* Praticar lógica de programação
* Aplicar estruturas condicionais e loops
* Trabalhar com listas e tuplas
* Organizar código em funções
* Criar um projeto apresentável para portfólio

---

## ⚙️ Funcionalidades

✅ Classificação da glicose:

* Normal (< 100 mg/dL)
* Pré-diabetes (100–125 mg/dL)
* Diabetes (≥ 126 mg/dL)

✅ Análise de alimentos com base no teor de açúcar

✅ Feedback claro para o usuário (✔️ ❌ ⚠️)

---

## 🧠 Tecnologias e conceitos

* Python 3
* Estruturas condicionais (`if`, `elif`, `else`)
* Laços de repetição (`for`)
* Funções
* Desempacotamento de tuplas
* Organização modular (separação de arquivos)
